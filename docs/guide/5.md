# 项目范围管理
范围管理的目标
* 要做什么
* 只做什么
范围包括
* 产品范围
* 项目范围

不同生命周期的范围管理对比
||预测型（瀑布）|迭代型/增量型|适应型（敏捷）|
|-|-|-|-|
|定义范围的时间|项目开始|每个迭代开始|随时|
|确认范围时间|项目结束|每个迭代结束|随时反馈|
|范围控制你文件|范围基准|版本配置文件|用户故事|
|发起人/客户参与|里程碑|周期性|持续性|

## 控制范围
### 范围蔓延
范围蔓延就是未经控制的产品或项目范围的扩大
* 镀金（项目团队主动增加的额外工作）
* 爬行（客户要求下未经正常的范围变更控制程序而出现的产品或项目范围的扩大）
