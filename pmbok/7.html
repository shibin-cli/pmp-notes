# 计算题相关

  • EV 挣值
  • AC 实际成本
  • PV 计划值
  • PB 项目预算
  • BAC 完工预算
  • EAC 完工估算
  • ETC 完工尚需估算

# 偏差

  • 进度偏差 SV = EV - PV
  • 成本偏差 CV = EV - AC
  • 进度绩效指数 SPI = EV / PC
  • 成本绩效指数CPI = EV / AC

# 完工估算

  • 按照之前的单价继续： EAC = BAC / CPI
  • 按照计划的单价：EAC = AC + (BAC - EV)
  • 进度和成本都按照之前的继续：EAC = AC +[(BAC-EV) / (CPI*SPI)]
  • 完工尚需绩效指数：
    • TCPI0 = (BAC-EV) / (BAC-AC)
    • TCPI1 = (BAC-EV) / (EAC-AC)
  • 完工偏差：VAC = EAC - BAC
